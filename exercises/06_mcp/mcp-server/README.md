# 簡易MCPサーバー（学習用）

……ふふ、これはMCPの仕組みを学ぶためのシンプルなサーバーだよ。

## 概要

学習用の最小限のMCPサーバーを実装するの。

## 機能

- 簡単なツールの提供
- リソースの提供
- プロンプトの提供

## セットアップ

```bash
cd mcp-server
pnpm install
pnpm build
pnpm dev
```

## MCPツール

### `get_calculation_history`
計算履歴を取得する

### `add_calculation`
新しい計算を追加する

### `clear_history`
履歴をクリアする

……ふふ、MCPの基本を理解するのにちょうどいい complexity だよ。
