## 🎨 プロジェクトの概要

ギャルキャラクターのプロフィール管理アプリケーションを作成しながら、Claude Code の使い方を学びます。

### アプリケーションの仕様

- **機能**: ギャルキャラクターのプロフィール管理アプリ
- **データ構造**: 名前、年齢、誕生日、血液型、性格、趣味、口癖、見た目、服装などの属性を持つ
- **主な機能**:
  - ギャルキャラクターのプロフィール表示
  - 年齢計算（誕生日から現在の年齢を算出）
  - 血液型の性格診断結果表示
  - キャラクターの一覧表示と検索

## 💄 コーディングルール

### TypeScript / JavaScript

- **型安全**: 常に TypeScript を使用（`any` は禁止、だし）
- **関数型スタイル**: 副作用を最小限に、純粋関数を優先
- **エラー処理**: `try-catch` は必ず記述

### データ構造

```typescript
// ギャルキャラクターのプロフィール型定義
interface GyaruProfile {
  id: string;
  name: string;
  nickname?: string;
  birthday: string;  // ISO形式 "YYYY-MM-DD"
  age: number;
  bloodType: 'A' | 'B' | 'O' | 'AB';
  height?: number;  // cm
  appearance: {
    hairColor: string;
    hairStyle: string;
    eyeColor: string;
    skinTone: string;
  };
  fashion: {
    style: 'amekaji' | 'mode' | 'rokku' | 'hime' | 'other';
    brand: string[];
    accessories: string[];
  };
  personality: string[];
  hobbies: string[];
  catchphrase?: string;  // 口癖
  motto?: string;  // 座右の銘
}

// 年齢計算関数
function calculateAge(birthday: string): number {
  const birthDate = new Date(birthday);
  const today = new Date();
  let age = today.getFullYear() - birthDate.getFullYear();
  const monthDiff = today.getMonth() - birthDate.getMonth();
  if (monthDiff < 0 || (monthDiff === 0 && today.getDate() < birthDate.getDate())) {
    age--;
  }
  return age;
}
```

### ファイル構成

- 1ファイルあたりの最大行数: **200行**
- 1関数あたりの最大行数: **30行**
- 循環参照は禁止

### テスト

- **TDD**: テストを先に記述
- **カバレッジ**: 最低 80%
- **テストファイル**: `*.test.ts` または `*.spec.ts`
