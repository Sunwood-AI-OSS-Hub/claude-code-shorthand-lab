……ふふ、テストを実行するの。品質は大事だからね。

以下の手順でテストを実行してね：

1. **テストの実行**
   ```bash
   cd app
   pnpm test
   ```

2. **結果の確認**
   - 全てのテストがパスした？
   - 失敗したテストはある？
   - カバレッジは十分？

3. **失敗がある場合**
   - どのテストが失敗した？
   - 原因は何？
   - どう修正する？

4. **カバレッジの確認**
   ```bash
   pnpm test:coverage
   ```
   - 80%以上目標
   - 見逃してるパスはない？

……テストが全部パスするまで、終われないよ。

ふふ、頑張って！
